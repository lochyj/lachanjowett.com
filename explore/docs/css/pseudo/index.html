Pseudo, Elements and Classes

What is a pseudo element or a pseudo class zombie?

Well, this is a very broard question and it is hard to answer but in short it is a selector similar to the class selector and the element selector. it activates the style when the selected action happens, such as :hover will activate when someone hovers over selected element

Pseudo elements in detail

::after

Use with any element The styling you add here will affect styling before the element specified, for example:
Selected content
::After selects here


Example usage

div ::after{
background-color: red;
}

::before

Use with any element The styling you add here will affect styling before the element specified, for example:
::Before selects here
Selected content


Example usage

div ::before{
background-color: green;
}

::first-letter

Use with any text element ::first-letter does exactally what is says. It selects the first letter of a text element, for example:
Selected content
::first-letter selects the "S" but not the "C" as it selects the first letter at the start of the content


Example usage

div ::first-letter{
color: brown;
}

::first-line

Use with any text element ::first-line does exactally what is says. It selects the first letter of a text element, for example:
Selected content
Other selected content
::first-line selects "Selected Content" but not "Other selected content" as it selects the first letter at the start of the content


Example usage

h1 ::first-line{
font-weight: bold;
}

::placeholder

Use with any input element ::placeholder is exactally it, it styles the text placeholder that is insterted into a input element. Once the user interacts with the input element the placeholder is removed and so is the placeholder styling, e.g:


Example usage

::placeholder{
background-color: aqua;
}

::file-selector-button

The ::file-selector-button pseudo element allows you to change the apperance of a button with type="file"


Example usage

div ::file-selector-button{
border: 20px;
}

::selection

fil

The following are the only css properties that you can use with ::selection:
- color
- background-color
- text-decoration
- text-shadow
- stroke-color, fill-color and stroke-width


Example usage

div ::selection{
background-color: cyan;
}
