Redirect with JS

How do I automatically redirect people from one page to another?

Well the answer is very very simple! All you need to do is follow the simple steps below.

Redirect:

1. Add this code to a script element in the body of the page:
setTimeout(() =>{window.location.replace("[page url]");}, [wait time])

2. Replace [page url] with the url of the page you want to redirect to

3. Replace [wait time] with the time you want it to take (in seconds) and then add 3 zeros to it, for example:
3 seconds would be 3000
or if you want it to imediately redirect then remove the time and the latter comma

Example usage:
<!DOCTYPE html>
<html lang="en">
<head>
</head>
<body>
<script>
setTimeout(() =>{window.location.replace("example.com");}, 2000)
</script>
</body>
</html>

Thats all!
